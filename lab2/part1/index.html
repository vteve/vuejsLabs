JavaScript Задания

Задание 1: Счетчик посещений

При загрузке страницы показывает, сколько раз вы ее посещали (данные сохраняются после перезагрузки).

Задание 2: Загрузка картинок (с порядком)

Запрашивает 5 ссылок на картинки и отображает их в порядке ввода.

Задание 3: Загрузка картинок (без порядка)

Запрашивает 5 ссылок на картинки и отображает их по мере загрузки.

Задание 4: Загрузка картинок (async/await) с порядком

Асинхронная загрузка картинок с сохранением порядка.

Задание 5: Загрузка картинок (async/await) без порядка

Асинхронная загрузка картинок без сохранения порядка.

Задание 6: Проверка IP-адресов

Проверяет 5 IP-адресов на принадлежность к запрещенным странам.
